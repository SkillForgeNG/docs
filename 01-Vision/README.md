# SkillForge Vision

This section contains the foundational strategic vision that guides all SkillForge decisions.

## Contents

- **VISION.md** — The long-term aspirational state; what SkillForge becomes
- **MISSION.md** — The core purpose; what SkillForge does for learners, instructors, and employers

## How to Use

Before making any architectural or product decision, ask:

1. **Does this decision move us toward our Vision?**
2. **Does this decision align with our Mission?**
3. **Does this decision embody our Principles?**

If the answer to all three is yes, proceed. If not, reconsider.

## Philosophy

SkillForge is not:
- A Learning Management System
- An online course platform
- A credentialing body
- A job board

SkillForge is:
- An AI-powered competency-based learning ecosystem
- Infrastructure for evidence-based skill verification
- A bridge between learning and labor markets
- A learner-owned, portable professional profile
